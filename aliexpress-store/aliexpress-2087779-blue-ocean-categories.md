## How to Add a New Sub-Category to This Store

1. Confirm it is NOT in Women's Clothing, Men's Clothing, or Novelty & Special Use L1 — those are permanently banned
2. All other L1 categories are pre-approved Blue Ocean — no further approval needed
3. Verify it maps to an authorized AliExpress template sheet
3. Download the template sheet from Seller Center
4. Add column mappings to `aliexpress/src/excel/column-mapping.ts`
5. Add to `CATEGORY_SHEET_MAP` in `aliexpress/src/models/product.ts`
6. Add 1688 Chinese search term to `1688_scrapper/src/config.ts`
7. Add to this file with status = ACTIVE
8. Tag new products with `store_id = '2087779'` and `platform = 'aliexpress'` in 1688_source DB
